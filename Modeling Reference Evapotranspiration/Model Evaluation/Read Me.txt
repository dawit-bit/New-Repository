This folder contains 4 files

1. 2017_2020_Hourly_xgb_Model_Eval.csv
              
   File:                                            Input data file in csv format 
   Content:                                         Weather and reference evapotranspiration data, eto, spanning 4 years (2017-2020), used for evaluation of the xgb model
                                                    The data types and units are described in the file: DataAcquisitionProcesssing.ipynb    

2. XGBoost_Predictions_EvalMode_2017_2020.ipynb 

   File type:                                       Jupiter Notebook  
   Content:                                         Python code for importing input data, loading it into pandas data frame, data preprocessing, loading the optimized_xgboost_eto file
                                                    (an output of the model training phase), and model evaluation 
   Description:                                     The xgboost model was evaluated using completely new (unseen) data 

3. eval_eto_predictions_output.csv

   File type:                                       Output data file in csv format.  
   Content:                                         Contains weather data, actual evapotranspiration data, and model predicted evapotranspiration 

4. eto_actual_vs_predicted.png 

   File type:                                       Output data chart in image file format
   Content:                                         Comparison of referenced evapotranspiration predicted by the xgboost model and observed
